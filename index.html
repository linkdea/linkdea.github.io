Frequently Asked Questions

How does this technologh work?

Our advanced signal processing algorithm is enabling the digital communication between devices by encoding a digital-key-signature into the audio. The key could be transfered via a physical speaker through the acoustic path, and it can be decoded via a separate microphone in the same room which holds the decoding counter part.

What kind of devices can be used with this technology ?

Any device equipped with a speaker can be used as a key/message transmiter, and any device equipped with a microphone can be used as a key/message receiever.

My devices are already connected via a wifi network, what is the advantage in using the technology for transmit key/messages ?

Wifi/mobile/bluetooth networks are using electromagnetic wave for data transmission. However, due to the nature of electromagnetic wave, it would be hard and costly to make the transmition to be location sensitive.
While, in using our acoustic-digital-communication technology, we can make the local device to be sensable if they are close to each other.
